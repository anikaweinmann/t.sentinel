DESCRIPTION

t.sentinel.mask is a GRASS GIS addon Python script to create a space time raster data set of cloud masks and shadow masks by using i.sentinel.mask.

EXAMPLE

t.sentinel.clouds input=s2 output=s2clouds nprocs=12

SEE ALSO

i.sentinel.mask, t.create, t.register
